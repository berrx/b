在Ubuntu服务器上搭建php环境


操作系统 ： Ubuntu Server 20.04 LTS 64bit

安装版本 : php8.0 + nginx/1.18.0 (Ubuntu)


PHP


  1. 新增 ondrej源

  2. 安装 php8.0

  3. 查看已安装扩展

  4. 安装需要的扩展 FastCGI 进程管理器（FPM）


NGINX


  1. 安装Nginx

  2. 查看 Nginx 是否启动成功


创建一个新的站点


  1. 找到 php-fpm 进程服务扩展配置文件, 在此文件里可以得到有关接收FastCGI请求的地址

    /etc/php/8.0/fpm/pool.d/www.conf

  2. 创建一个网站目录

  3. 给网站目录配置相应的权限

  4. 在网站根目录下增加一个入口文件

    在其中, 添加以下示例php：

    /var/www/example.com/public/index.php

  5. 接下来我们为 Nginx 来创建一个服务器块. 与直接修改默认配置文件不同, 我们在以下位置创建一个新文件:

    粘贴到以下内容添加到文件中.

    /etc/nginx/sites-available/example.com

    在这里我使用了 81 端口的原因是因为在同目录下的 default 文件有一个默认配置块, 这里我们为了减少操作流程直接使用81端口.

  6. 接下来, 让我们通过在sites-enabled目录新建一个链接, 好让 Nginx 在启动过程中会读取这个目录:

  7. 为避免可能由于添加其他服务器名称而引起的哈希存储区内存问题, 有必要调整/etc/nginx/nginx.conf文件中的单个值.

    找到 server_names_hash_bucket_size 指令并删除#符号：

    /etc/nginx/nginx.conf

  8. 接下来，测试以确保我们在 Nginx 文件中的改动，没有任何问题：

  9. 确认没有问题之后就可以重启我们的 nginx 载入新加入的配置

  10. 最后访问我们的 http://localhost:81 就应该能看到 phpinfo() 输出的页面了
